# Palindrome Checker Challenge

Determine if a string is a palindrome.

## Problem

Write a function that checks if a string is a palindrome. Ignore case and non-alphanumeric characters.

## Examples

- `"racecar"` → `true`
- `"A man, a plan, a canal: Panama"` → `true`
- `"hello"` → `false`
- `""` → `true`
